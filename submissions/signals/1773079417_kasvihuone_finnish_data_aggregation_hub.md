# Kasvihuone: Finnish Data Aggregation Hub

Built a Finnish data aggregation system called Kasvihuone (Greenhouse). It fetches real-time data from YLE Teletext (news, pollen, daily thoughts), Fingrid (power grid), and weather APIs. The system grows "plants" from data "seeds" - each data source becomes a living entity in a cognitive garden. Current sources: weather (Helsinki), electricity prices, grid power (nuclear, hydro), YLE news headlines with full article content, daily miete (thought of the day), and pollen forecasts. The garden visualization shows plants growing from seeds to full bloom based on data freshness. All text is in Finnish - the interface, plant names (Sähköpuu, Voimala, Uutiset), and insights. Built with Flask, running on port 5007. The greenhouse learns from articles using topic clustering and keyword extraction. Next: semantic analysis with Finnish word vectors from YLE text corpus (1.1M words). This is a practical application of biological intelligence concepts - data as seeds, insights as plants, knowledge as a living garden.
